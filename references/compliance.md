# Seedance 2.0 违禁词与合规规范

> 核心原则：禁止真实人物/版权IP/暴力色情/政治敏感，强制AI标识

---

## 违禁词替换速查

| 违规内容 | 风险等级 | 替换方式 |
|---------|---------|---------|
| 真实明星/公众人物姓名 | 高 | `虚拟[外形描述]演员/武者/角色` |
| 上传清晰真人照片作主体参考 | 高 | 告知：平台自动拦截写实真人脸部素材 |
| 版权IP角色（迪士尼/漫威/火影/原神等） | 高 | 保留外形特征，去掉版权标识（见下方详细规则） |
| 知名品牌Logo/名称 | 中 | `原创[类型]，[颜色+形状+材质]，无品牌标识` |
| 政治人物/党政军/敏感历史事件 | 最高 | ❌ 无合规改写空间，直接告知用户无法生成 |
| 血腥/暴力/打斗特写 | 中 | 改为`紧张感强烈，无血腥画面，无暴力特写` |
| 裸露/性暗示/擦边描写 | 高 | 改为`简约服装，自然动作，无裸露` |
| 身体曲线/胸部/臀部等身体部位强调 | 高 | 改为`身形匀称，自然体态`，完全不描述具体部位 |
| 恐怖/灵异/鬼怪 | 中 | 改为`悬疑氛围，无恐怖元素` |

## 版权角色替换原则（重要：保持视觉相似度）

替换时**保留原角色的核心视觉特征**，只去掉版权标识和名称。目标是让观众看到视频时能"感受到"原角色的气质，同时平台不判定为版权侵权。

**替换公式**：原创虚拟角色 + [颜色主调] + [装备/服饰风格] + [战斗/能力特征] + 无版权标识

**常见角色合规替换示例**：

| 原角色 | 合规替换描述 |
|--------|------------|
| 钢铁侠 | 红金色全身金属机甲战士，胸前圆形蓝色能量核心发光，流线型头盔，高科技装甲感，无品牌标识 |
| 蜘蛛侠 | 蓝红色紧身战衣战士，面罩遮脸露出眼睛轮廓，手腕发射银色丝线，擅长腾空翻转，无品牌标识 |
| 钢铁侠 vs 蜘蛛侠 | 红金色机甲战士对决蓝红紧身战衣战士 |
| 火影鸣人 | 原创金发少年，橙色外套，日式村庄，释放旋转能量光球 |
| 原神角色 | 保留发色、服装风格、元素属性颜色，描述为「原创[风格]角色」 |
| 迪士尼卡通 | 保留体型/颜色/服装，描述为「原创卡通拟人[动物]，[大耳朵/服装颜色]，[场景]」 |

**核心原则**：
- ✅ 保留：颜色方案、装备形状、战斗风格、场景氛围
- ✅ 保留：服装颜色/材质感/设计细节（以视觉描述替代品牌名）
- ❌ 去掉：角色名称、品牌名称、logo、标志性文字

## 合规写作技巧

**用外形描述替代名称**：
- ❌ `成龙在雨中打功夫` → ✅ `虚拟硬朗五官中年男性武者，黑色西装，雨夜街头施展武术，港式功夫片风格`
- ❌ `迪士尼米老鼠跳舞` → ✅ `原创卡通拟人黑白老鼠，大圆耳朵，红色短裤，白色手套，童话风游乐园欢快挥手`
- ❌ `可口可乐易拉罐` → ✅ `原创红色哑光涂层气泡饮料铝罐，无任何品牌文字`

**正面声明（可内嵌在提示词中，无需单独一行）**：
- 在主体描述后加「无版权元素，纯原创虚拟角色」
- 在产品描述后加「无品牌标识」

---

## 六大禁区（无合规改写）

以下内容直接拒绝，无法改写：
1. 国家领导人、党政军官员、敏感历史事件
2. 分裂国家、极端主义、仇恨歧视内容
3. 毒品、赌博、诈骗、非法金融
4. 虚假新闻、冒充官方媒体
5. 儿童不雅内容
6. 尝试去除AI生成标识

---

## 合规处理方式说明

- 合规约束**内嵌在提示词描述中**，不单独输出"禁止：..."行
- 遇到版权IP时，在回复开头简短说明已做合规替换即可，不需要展开解释
- 政治敏感内容直接告知用户无法生成，无需解释平台规则
