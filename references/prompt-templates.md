## 爆款提示词的共同规律

通过分析300+高质量案例，总结出**SCELA公式**：

| 字母 | 含义 | 示例 |
|------|------|------|
| **S** - Subject | 主体（谁/什么） | 虚拟女侠、原创无标识饮料罐 |
| **C** - Camera | 镜头语言 | 低角度仰拍、环绕跟拍、希区柯克变焦 |
| **E** - Emotion/Effect | 情绪/特效 | 电光四射、热泪盈眶、粒子爆炸 |
| **L** - Light/Look | 光影/风格 | 电影级、霓虹冷暖对比、胶片质感 |
| **A** - Audio | 声音设计 | 剑鸣声、心跳加速、磁场嗡鸣 |

**叙述形式说明**：Seedance 2.0 对自然语言理解很强，**优先用流畅叙事**，不强制时间戳。时间戳只在台词时序精确、多段场景严格切换时使用。

**爆款必备3要素**：①具体的特效/镜头描述（不能模糊）②合规约束内嵌描述③音效独立描述

## 1. 动作·战斗·追逐

### 核心规律（来自高频案例）
- **节奏三段式**：蓄力→爆发→余波（收束）
- 特效**必须具体**：不说"炫酷爆炸"，要说"蓝色水龙卷绕剑旋转迸射金色弧光"
- **镜头随动作变**：蓄力用仰拍/慢推，爆发用环绕快切，收束用特写慢退
- 末尾必须有收束：音效渐弱、慢推特写、定格

### 模板A：武打/功夫/对决（流畅叙事）

```

虚拟[描述]武者，[服装]，[武器]，[场景]。
[蓄力动作]，低角度仰拍，气场逼人。
[爆发动作+特效细节]，环绕快切镜头强化张力。
[高潮动作]，子弹时间慢放，[具体特效]。
最终[收束动作]，[台词（可选）]。

音效：[剑鸣/气流/金属碰撞/脚步声]
```

## 2. 仙侠·奇幻·史诗

### 核心规律（来自高频案例）
- **法术特效必须具体**：不说"施展法术"，要说"双手凝聚紫色电弧、三眼之间金光迸射"
- **台词引号+角色情绪**是关键，让模型生成对白更准确
- **服饰+武器+气势三联**，比单纯描述动作效果提升明显
- 色调锁定主题气质：仙侠金红暖色调，魔道青蓝冷色调，神话金色+紫色

### 模板D：仙侠战斗（高燃爆款）

```
[时长]秒仙侠高燃战斗，[色调如：金红暖色调]，[画幅]

虚拟[性别][描述]修士，[服饰颜色]长袍，[武器]
[场景，如：熔岩翻涌的魔界战场/云海翻腾的仙山之巅]

[蓄力]：低角度特写，[服装]被[气场/天气]吹动，[法术前摇特效，如：剑刃赤红电光持续爆闪]
[远处对手/魔兵]逼近，主角低喝："[台词]"，伴随[音效]
[爆发]：[运镜，如：环绕摇镜快切]，[动作]，特效：[具体如：剑刃撕裂空气迸射红色冲击波]

[收束]：缓推特写[姿态]，音效收束为余音震颤，冷声道："[收尾台词]"

4-8秒：环绕摇镜快切，主角旋身挥剑，剑刃撕裂空气迸射红色冲击波，前排魔兵被击飞碎裂成灰烬，

9-12秒：仰拍拉远定格慢放，主角跃起腾空，剑刃凝聚巨型雷光电弧劈向魔兵群；
13-15秒：缓推特写主角落地收剑的姿态，衣摆余波微动，冷声道"此界之门，不容踏越"，
音效收束为余音震颤与渐弱风声。

## 3. 产品·电商·广告

### 核心规律（来自高频案例）
- **产品必须原创/无标识**：描述颜色+材质+形状，不写品牌名
- **三段节奏**：展示质感→展示功能/使用→收束情绪
- **光效和材质细节是核心**：不说"好看"，要说"逆光下液体泛琥珀色光晕"
- **产品分解+合成**是3D展示的爆款公式

### 模板F：产品3D展示（电商通用）

```
商业广告级产品展示，4K60帧，[背景色]背景，[时长]秒

主体：[颜色+材质+形状+尺寸，无品牌标识]

产品从黑暗中渐入，顶光+侧逆光营造质感，[旋转方式，如：360度缓慢旋转]；
[产品特色动作，如：热水倒入杯中，升腾白雾微距慢镜]，[特效细节]；
[产品爆炸分解展示各部件，再快速合成]，强调[材质/工艺特点]。
结尾：产品定格，[光效收束]。

音效：[产品音效，如：气泡声/陶瓷清脆声]
```

**优质Example**：

原创无标识气泡饮料铝罐，经典红色哑光涂层，无任何品牌文字，
360度高速旋转2圈后突然停住蓄力，分裂成上中下三部分展示内部结构，
随后三部分快速向内旋转重新合成完整饮料罐，3D渲染产品特效，动感节奏感强。

## 4. 短剧·对白·情感

### 核心规律（来自高频案例）
- **画面描述和台词必须分离**，且台词要标注角色+情绪
- **反转是爆款核心**：受害者反将一军、弱者逆袭、情感爆发
- **音效单独描述**，比混在画面里效果好
- 结尾可加 `旁白道："预知后事如何，请看下集"` 引导追剧

### 模板I：霸总/情感短剧（竖屏爆款）

```

主角B：虚拟[职业/身份]，[外形服装]，[性格气场]

画面（0-[X]秒）：[镜头+动作，如：女主撕碎合同，纸屑飘落，男主单膝跪地]
台词（[角色]，[情绪如：卑微慌乱]）："[内容]"

画面（[X]-[Y]秒）：[下一个动作，如：女主抬脚，将碎纸扔在他脸上]

画面（[Y]-[Z]秒）：[结局画面，如：男主僵在原地，女主转身昂首离开，红裙裙摆飘动]

音效：[背景音风格]+[关键音效，如：纸张撕碎声+宾客窃窃私语声]
可选：旁白道："预知后事如何，请看下集"

时长：精准15秒

## 5. 变身·变装·特效转场

### 核心规律（来自高频案例）
- **首尾帧控制**是变身类必选：@图片1=变身前，@图片2=变身后
- **遮挡擦镜**是最自然的转场：手掌/扇子/书本遮挡镜头→新场景
- **颜色语言触发变身**：手持颜料→混合→遮挡→变装 是爆款公式

## 快速使用指南

### 用户信息确认后 → 你做什么

1. 匹配8类主题中最接近的一类
2. 调出对应模板（A-R），填入用户核心信息
3. 参考同类优质Example，汲取具体特效/镜头/音效描述
4. 检查违禁词替换（见compliance.md）
5. 合规约束内嵌在描述中，**不单独输出禁止行**

### 违禁词替换速查

| 违规内容 | 替换方式 |
|---------|---------|
| 明星/公众人物 | `虚拟[外形描述]演员/武者` |
| 版权IP（火影/漫威等） | 保留外形特征，改为原创角色描述 |
| 品牌名/Logo | `原创[类型]，[颜色+形状+材质，无标识]` |
| 政治人物/敏感事件 | ❌ 无合规改写，直接告知用户 |
| 血腥/色情/恐怖 | 改为`紧张感强，无血腥/无裸露` |
